WeatherPy Project

My assignment was to prove that temperatures increase when moving closer to the equator. Even though that's a 'duh' statement, how does one prove it with data?

This was a fun assignment given to me by Kansas University's Data Analytics Bootcamp.

Click for a free hug!

The jist of the assignment: Firstly, I was to pull in data via an API call to the Open Weather Database. I randomly picked 500+ latitude and longitude combinations, of cities around the world, by using NumPy. The .json data that came back from the API call contained temperatures, humidity counts, windspeeds, amongst others. Secondly, I placed the data into a Pandas DataFrame. Finally, I was able to use Matplotlib to do plots within my Jupyter Notebook. Have fun here, and take care!
